# 애드센스 승인용 필수 정적 페이지 체크리스트 (초안)

작성: CS팀 | 상태: 항목 제안 초안, 실제 법률 문구 미작성 | 최종수정: 2026-07-08

> 주의: 아래는 "어떤 섹션/항목이 필요한지"에 대한 체크리스트일 뿐, 실제 법률적 확정 문구가 아닙니다.
> 법률/개인정보보호법/전자상거래법 등 관련 확정 문구는 반드시 **전문가 검토 필요** 표시된 항목을 거쳐야 합니다.

## /privacy (개인정보처리방침) — 전문가 검토 필요 (전체)
- [ ] 수집하는 개인정보 항목 (예: 쿠키, 접속 로그, IP 등) — 전문가 검토 필요
- [ ] 개인정보 수집 목적
- [ ] 보유 및 이용 기간
- [ ] 제3자 제공 여부 (Google AdSense, Google Analytics 등 서드파티 명시) — 전문가 검토 필요
- [ ] 쿠키 사용 안내 및 광고 개인 최적화(맞춤 광고) 관련 고지 — Google 정책 요구사항, 전문가 검토 필요
- [ ] 이용자의 권리(열람/정정/삭제 요청 방법) 및 처리 절차 — 전문가 검토 필요 (개인정보보호법 준수)
- [ ] 개인정보 보호책임자/담당자 연락처
- [ ] 시행일자 및 개정 이력
- [ ] 미성년자 개인정보 처리 관련 문구 (해당 시) — 전문가 검토 필요

## /terms (이용약관) — 전문가 검토 필요 (전체)
- [ ] 목적 및 적용 범위
- [ ] 서비스 이용 조건 (무료/유료 여부, 회원가입 필요 여부)
- [ ] 계산 결과에 대한 면책 조항 (참고용이며 법적/재무적 책임 없음 명시) — 전문가 검토 필요
- [ ] 지적재산권 (콘텐츠 저작권)
- [ ] 이용자의 의무 및 금지행위
- [ ] 서비스 변경/중단 고지
- [ ] 광고 게재 관련 고지 (제3자 광고 포함, 사이트 운영 방식)
- [ ] 준거법 및 관할법원 — 전문가 검토 필요
- [ ] 시행일자

## /contact (문의처)
- [ ] 문의 접수 방법 (이메일 양식 등)
- [ ] 응답 소요 시간 안내 (예: 영업일 기준 n일 이내)
- [ ] 사업자 정보 표기 여부 확인 (사업자등록 여부에 따라 상이) — 전문가 검토 필요
- [ ] 개인정보 관련 별도 문의 채널 안내 (담당자/이메일)
- [ ] 버그 제보용 안내 문구 (계산 오류 제보 시 필요한 정보 안내)

## /about (사이트 소개)
- [ ] 사이트 운영 목적/취지 소개
- [ ] 제공 계산기 종류 개요
- [ ] 계산 결과의 참고용 성격 안내 (면책 성격, 간단 버전)
- [ ] 운영자/팀 소개 (선택)
- [ ] 문의 페이지로의 링크 연결

## 공통 (전 페이지)
- [ ] 최종 수정일 표기
- [ ] 모바일 가독성 확보 (레이아웃은 개발팀 담당)
- [ ] Google AdSense 프로그램 정책 요구사항 충족 여부 최종 확인 — 전문가/마스터 팀장 검토 필요
