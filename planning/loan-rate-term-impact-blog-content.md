## (b) 자가 점검 체크리스트

### 타입 유니온 준수
- [x] `heading`: text만 사용 (level 필드 없음) — 전 heading 확인
- [x] `paragraph`: text만
- [x] `list`: ordered(boolean)+items(string[])만 — SET1/SET2 두 개, 둘 다 ordered:false
- [x] `callout`: variant는 "info"/"warning"만 사용 (info 1개, warning 1개)
- [x] `calculatorCta`: slug만 사용, label 등 추가 필드 없음 (3개: loan-interest / loan-prepayment-fee / dsr)
- [x] 마크다운 표 없음 → heading+list(ordered:false) 구조로만 수치 표현
- [x] BlogPost 필드 전부 존재: slug/title/description/category/tags/publishedDate/relatedCalculatorSlugs/body
- [x] tags 정확히 5개
- [x] publishedDate "2026-07-17"
- [x] category "loan"
- [x] relatedCalculatorSlugs 3개 모두 calculators.ts 실재 live slug (Grep 확인: loan-interest-calculator L240, loan-prepayment-fee L387, dsr-calculator L552) — 형제글과 대칭

- [x] SET2 5행(12/24/36/60/120개월) 월상환액·총이자·총상환액 전부 앵커 표와 일치
